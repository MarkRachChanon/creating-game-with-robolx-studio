# PART 1: พื้นฐานและการเริ่มต้น

## บทที่ 1: รู้จักกับ Roblox และ Roblox Studio

### 1.1 Roblox คือแพลตฟอร์มอะไร?

**ทฤษฎี:**

**Roblox** คือแพลตฟอร์มออนไลน์ที่ช่วยให้ผู้ใช้สามารถ:
- **เล่นเกม** ที่ผู้อื่นสร้างขึ้น (มีเกมมากกว่า 40 ล้านเกม)
- **สร้างเกม** ของตัวเองโดยไม่ต้องเขียนโค้ดมากนัก
- **แชร์และเผยแพร่** ผลงานของตัวเองให้คนทั่วโลกได้เล่น
- **สร้างรายได้** จากเกมที่สร้าง ผ่านระบบ Robux (สกุลเงินในเกม)

**สถิติที่น่าสนใจ:**
- ผู้ใช้งานมากกว่า 200 ล้านคนต่อเดือน
- นักพัฒนาหลายคนสร้างรายได้หลักล้านบาทต่อปี
- เป็นแพลตฟอร์มที่ใช้ได้ทั้ง PC, Mobile, Xbox และ VR

**Roblox Studio คืออะไร?**

**Roblox Studio** คือโปรแกรมที่ใช้สร้างเกมบน Roblox โดยมีคุณสมบัติ:
- ✅ **ฟรี 100%** - ดาวน์โหลดและใช้งานได้ไม่เสียค่าใช้จ่าย
- 🎮 **เหมาะกับมือใหม่** - มี Interface ที่ใช้งานง่าย
- 💻 **รองรับการเขียนโค้ด** - ใช้ภาษา Lua (ง่ายกว่า C++ หรือ Java)
- 🌐 **เผยแพร่ได้ทันที** - สร้างเสร็จกดปุ่มเดียวเกมขึ้นให้เล่นได้เลย

---

### 1.2 การติดตั้ง Roblox Studio

**ขั้นตอนการติดตั้ง (Windows):**

#### ขั้นตอนที่ 1: สมัครบัญชี Roblox

1. เปิดเว็บเบราว์เซอร์ (Chrome, Edge, Firefox)
2. ไปที่ https://www.roblox.com
3. คลิกปุ่ม **Sign Up** (สมัครสมาชิก)
4. กรอกข้อมูล:
   - **Username** (ชื่อผู้ใช้ - ห้ามซ้ำกับคนอื่น)
   - **Password** (รหัสผ่าน - อย่างน้อย 8 ตัวอักษร)
   - **วันเกิด** (ต้องกรอกให้ถูกต้อง)
   - **เพศ** (Optional)
5. คลิก **Sign Up**

> ⚠️ **หมายเหตุ:** เก็บ Username และ Password ไว้ให้ดี จะใช้ตอนล็อกอินเข้า Roblox Studio

#### ขั้นตอนที่ 2: ดาวน์โหลด Roblox Studio

1. ล็อกอินเข้าบัญชี Roblox ที่เพิ่งสร้าง
2. ไปที่เมนูบนสุด คลิก **Create** (สร้าง)
3. หน้าเว็บจะเปิดไปที่ https://create.roblox.com
4. คลิกปุ่ม **Download Studio**
5. ไฟล์ `RobloxStudio.exe` จะถูกดาวน์โหลด

#### ขั้นตอนที่ 3: ติดตั้งโปรแกรม

1. เปิดไฟล์ `RobloxStudio.exe` ที่ดาวน์โหลดมา
2. รอโปรแกรมติดตั้งอัตโนมัติ (ประมาณ 2-5 นาที)
3. เมื่อติดตั้งเสร็จ Roblox Studio จะเปิดขึ้นมาเอง
4. ล็อกอินด้วยบัญชีที่สมัครไว้

> 💾 **ขนาดโปรแกรม:** ประมาณ 400-500 MB  
> ⏱️ **เวลาติดตั้ง:** 2-5 นาที (ขึ้นอยู่กับความเร็วอินเทอร์เน็ต)

#### ขั้นตอนที่ 4: ตรวจสอบการติดตั้ง

หลังจากเปิด Roblox Studio จะเห็นหน้าต่าง **Start Page** ที่มี:
- **New** - สร้างโปรเจกต์ใหม่
- **Recent** - โปรเจกต์ที่เปิดล่าสุด
- **My Games** - เกมที่คุณสร้างไว้

### 1.3 ทำความรู้จักกับ Interface ของ Roblox Studio

เมื่อเปิด Roblox Studio ขึ้นมา จะเห็นส่วนต่างๆ ดังนี้:

```
┌─────────────────────────────────────────────────────────────┐
│  [Home] [Model] [Test] [View] [Plugins]    <- Ribbon Menu   │
├─────────────────────────────────────────────────────────────┤
│ Explorer │                                │ Properties       │
│  ├─Workspace                              │  Name: Part      │
│  ├─Players                                │  Size: 4,1,2     │
│  ├─Lighting         3D Viewport            │  Color: Gray     │
│  └─...              (พื้นที่สร้างเกม)      │  Material: Plastic│
│                                             │                  │
│                     กล้องมุมมอง            │                  │
│          Part                               │                  │
│         ┌────┐      ตัวละคร                │                  │
│         │    │         👤                   │                  │
│         └────┘                              │                  │
│  ─────────────────  พื้นฐาน  ──────────── │                  │
└─────────────────────────────────────────────────────────────┘
│  Output (แสดงข้อความ Error, ผลลัพธ์)                         │
└─────────────────────────────────────────────────────────────┘
```

#### ส่วนประกอบสำคัญ:

**1. Ribbon Menu (แถบเมนูด้านบน)**
- **Home Tab** - เครื่องมือพื้นฐาน (ย้าย, หาย, ลบ)
- **Model Tab** - สร้าง Part, Terrain, เพิ่ม Effect
- **Test Tab** - ทดสอบเกม (เล่นเกม, หยุด)
- **View Tab** - เปิด/ปิดหน้าต่างต่างๆ
- **Plugins Tab** - เครื่องมือเสริม

**2. Explorer (รายการวัตถุ)**
- แสดงโครงสร้างของเกมทั้งหมด
- มี Folder สำคัญ:
  - **Workspace** - วัตถุที่อยู่ในเกม (Part, Model, NPC)
  - **Players** - ผู้เล่นที่เข้ามาในเกม
  - **Lighting** - แสงสว่างในเกม
  - **ReplicatedStorage** - เก็บ Script/Asset ที่ใช้ร่วมกัน
  - **StarterPlayer** - ตั้งค่าตัวละครผู้เล่น

**3. Properties (คุณสมบัติ)**
- แสดงและแก้ไขคุณสมบัติของวัตถุที่เลือก
- เช่น สี, ขนาด, ตำแหน่ง, วัสดุ

**4. 3D Viewport (พื้นที่ทำงานหลัก)**
- พื้นที่สร้างและจัดวาง Part
- ใช้เมาส์ควบคุมมุมมอง
- วางตัวละคร (Character) และวัตถุต่างๆ

**5. Output (หน้าต่างแสดงผล)**
- แสดงข้อความ Error
- แสดงผลจากคำสั่ง `print()`
- Debug โค้ด

### 1.4 การควบคุมมุมมอง (Camera Control)

#### การใช้เมาส์:

| การกระทำ | วิธีทำ |
|---------|--------|
| **หมุนมุมมอง** | คลิกขวาค้าง + ลากเมาส์ |
| **เลื่อนมุมมอง** | กดปุ่มกลาง (Scroll) ค้าง + ลากเมาส์ |
| **ซูมเข้า-ออก** | ใช้ Scroll Wheel |
| **โฟกัสที่วัตถุ** | เลือก Part แล้วกด **F** |

#### การใช้คีย์บอร์ด:

| ปุ่ม | การทำงาน |
|------|---------|
| **W, A, S, D** | เลื่อนกล้องไปทิศทางต่างๆ |
| **Q** | ลงต่ำ |
| **E** | ขึ้นสูง |
| **Shift + W/A/S/D** | เคลื่อนที่เร็วขึ้น |
| **F** | โฟกัสที่วัตถุที่เลือก |

### 1.5 สร้างโปรเจกต์แรก

#### ปฏิบัติ: สร้างเกมง่ายๆ ด้วย Template

**ขั้นตอนที่ 1: เลือก Template**

1. เปิด Roblox Studio
2. ที่หน้า **Start Page** เลือก **New**
3. เลือก Template:
   - **Baseplate** - แผ่นพื้นเปล่าๆ (เหมาะกับการเริ่มต้น)
   - **Flat Terrain** - แผ่นพื้นดินขนาดใหญ่
   - **Classic Baseplate** - พื้นเก่าแบบ Roblox สมัยก่อน

4. คลิก **Baseplate** แล้วกด **Create**

**ขั้นตอนที่ 2: ทำความเข้าใจโครงสร้าง**

เมื่อเปิดโปรเจกต์ขึ้นมา จะเห็น:
- **SpawnLocation** (แผ่นสีฟ้า) - จุดเกิดของผู้เล่น
- **Baseplate** (แผ่นสีเทาขนาดใหญ่) - พื้นของเกม
- ตัวละคร (หุ่นสีขาว) - แสดงตำแหน่งตัวละครผู้เล่น

**ขั้นตอนที่ 3: ทดสอบเกม**

1. ไปที่แท็บ **Test** บน Ribbon Menu

3. เกมจะเริ่มทำงาน คุณจะสามารถ:
   - เดินได้ด้วย **W, A, S, D**
   - กระโดดด้วย **Space**
   - หมุนมุมมองด้วยเมาส์

### 1.6 การบันทึกโปรเจกต์

#### วิธีที่ 1: บันทึกลงคอมพิวเตอร์ (Local)

1. คลิก **File** > **Save to File**
2. ตั้งชื่อไฟล์ เช่น `MyFirstGame.rbxl`

> 💾 **นามสกุลไฟล์:** `.rbxl` (Roblox Place File)

4. คลิก **Create**

> ☁️ **ข้อดี:** เข้าถึงได้จากเครื่องไหนก็ได้ ไม่กลัวไฟล์หาย

## บทที่ 2: Part และการสร้างวัตถุพื้นฐาน

### 2.1 Part คืออะไร?

**ทฤษฎี:**

**Part** คือหน่วยพื้นฐานที่สุดในการสร้างเกม Roblox ทุกอย่างในเกมล้วนประกอบด้วย Part เช่น:
- **อาคาร** - สร้างจากหลาย Part ต่อกัน
- **พื้นดิน** - เป็น Part ขนาดใหญ่แบนราบ
- **ประตู** - Part ที่มีความโปร่งแสง
- **ขั้นบันได** - Part วางซ้อนกัน

คุณสมบัติของ Part:
- มีรูปร่าง (Shape) - ลูกบาศก์, ทรงกลม, ทรงกระบอก
- มีวัสดุ (Material) - พลาสติก, โลหะ, ไม้, หิน
- มีสี (Color)
- มีขนาด (Size)
- มีน้ำหนักและฟิสิกส์

---

### 2.2 การสร้าง Part

#### ปฏิบัติ: สร้าง Part แรก

**ขั้นตอนที่ 1: เพิ่ม Part**

1. ไปที่แท็บ **Home** หรือ **Model**
2. คลิกปุ่ม **Part** (รูปลูกบาศก์)
3. Part จะปรากฏตรงกลางหน้าจอ

**ขั้นตอนที่ 2: เลือก Part**

- คลิกที่ Part ใน Viewport
- Part ที่เลือกจะมีกรอบสีฟ้าล้อมรอบ
- Properties จะแสดงคุณสมบัติของ Part นั้น

---

### 2.3 การแก้ไขคุณสมบัติ Part

#### ปฏิบัติ: ปรับแต่ง Part

**1. เปลี่ยนขนาด (Size)**

ในหน้าต่าง **Properties** หา **Size**:
- **X** - ความกว้าง (ซ้าย-ขวา)
- **Y** - ความสูง (ขึ้น-ลง)
- **Z** - ความลึก (หน้า-หลัง)

ตัวอย่าง:
```
Size = 10, 1, 10  → แผ่นพื้นขนาดใหญ่
Size = 1, 10, 1   → เสาสูง
Size = 4, 0.2, 2  → ขั้นบันได
```

**2. เปลี่ยนสี (Color)**

1. ในหน้าต่าง Properties หา **BrickColor**
2. คลิกที่สี แล้วเลือกสีที่ต้องการ
3. หรือใช้ **Color3** เพื่อเลือกสีแบบละเอียด (RGB)

**3. เปลี่ยนวัสดุ (Material)**

1. ในหน้าต่าง Properties หา **Material**
2. เลือกวัสดุ:
   - **Plastic** - พลาสติก (Default)
   - **Wood** - ไม้
   - **Metal** - โลหะ
   - **Concrete** - คอนกรีต
   - **Grass** - หญ้า
   - **Glass** - กระจก (โปร่งแสง)

**4. เปลี่ยนรูปร่าง (Shape)**

1. ในหน้าต่าง Properties หา **Shape**
2. เลือก:
   - **Block** - ลูกบาศก์ (Default)
   - **Ball** - ทรงกลม
   - **Cylinder** - ทรงกระบอก
   - **Wedge** - รูปลิ่ม (สำหรับทำหลังคา)

---

### 2.4 เครื่องมือจัดการ Part

#### ปฏิบัติ: ใช้ Tools เบื้องต้น

**1. Select Tool (เลือก)**
- ใช้: คลิกเลือก Part

**2. Move Tool (ย้าย)**
- กด: **Ctrl + 2** หรือ **Home > Move**
- จะเห็นลูกศร 3 แกน (X=แดง, Y=เขียว, Z=น้ำเงิน)
- ลากแกนที่ต้องการย้าย

**3. Scale Tool (ขยาย/ย่อ)**
- กด: **Ctrl + 3** หรือ **Home > Scale**
- จะเห็นจุดสีเหลืองที่มุม Part
- ลากจุดเพื่อขยาย/ย่อ

**4. Rotate Tool (หมุน)**
- กด: **Ctrl + 4** หรือ **Home > Rotate**
- จะเห็นวงกลม 3 แกน
- ลากเพื่อหมุน Part

**คำสั่งพื้นฐาน:**
- **Ctrl + D** - Duplicate (ทำสำเนา)
- **Delete** - ลบ Part
- **Ctrl + Z** - Undo (ย้อนกลับ)
- **Ctrl + Y** - Redo (ทำซ้ำ)

---

### 2.5 การล็อค Part (Anchored)

**ทฤษฎี:**

**Anchored** คือคุณสมบัติที่กำหนดว่า Part จะ:
- ✅ **Anchored = true** → ตรึงที่ไม่ขยับ (ไม่ตกลงมา)
- ❌ **Anchored = false** → มีฟิสิกส์ (ตกลงมาได้)

**การใช้งาน:**
- **พื้น, ผนัง, อาคาร** → ควร Anchored = true
- **ลูกบอล, กล่องที่ต้องเคลื่อนที่** → Anchored = false

#### ปฏิบัติ: ทดสอบ Anchored

1. สร้าง Part ใหม่
2. ในหน้าต่าง Properties หา **Anchored**
3. ติ๊ก ✅ เพื่อเปิด Anchored
4. กด **Play** เพื่อทดสอบ → Part จะไม่ตก
5. ปิด Anchored (ไม่ติ๊ก)
6. กด **Play** อีกครั้ง → Part จะตกลงมา

---

### 2.6 โปรเจกต์ฝึกปฏิบัติ: สร้างบ้านหลังแรก

#### ปฏิบัติ: สร้างบ้านง่ายๆ

**ขั้นตอนที่ 1: สร้างพื้น**

1. สร้าง Part ใหม่
2. ตั้งค่า:
   - Size: `20, 1, 20`
   - Material: `Concrete`
   - Color: สีเทา
   - Anchored: ✅
3. ย้ายลงพื้น Baseplate

**ขั้นตอนที่ 2: สร้างผนัง**

1. สร้าง Part ใหม่
2. ตั้งค่า:
   - Size: `20, 10, 1` (ผนังด้านหน้า)
   - Material: `Brick`
   - Color: สีน้ำตาล
   - Anchored: ✅
3. ใช้ **Move Tool** วางติดกับพื้น
4. กด **Ctrl + D** ทำสำเนา 3 ครั้ง
5. ใช้ **Rotate Tool** หมุนผนังด้านข้างและด้านหลัง

**ขั้นตอนที่ 3: สร้างหลังคา**

1. สร้าง Part ใหม่
2. เปลี่ยน Shape เป็น **Wedge**
3. ตั้งค่า:
   - Size: `22, 5, 10`
   - Material: `Wood Planks`
   - Color: สีแดง
   - Anchored: ✅
4. วางบนผนัง และทำสำเนาอีก 1 ชิ้นวางอีกด้าน

**ขั้นตอนที่ 4: สร้างประตู**

1. สร้าง Part ใหม่
2. ตั้งค่า:
   - Size: `4, 7, 0.5`
   - Material: `Wood`
   - Color: สีน้ำตาลเข้ม
   - Anchored: ✅
3. วางกลางผนังด้านหน้า

**ขั้นตอนที่ 5: ทดสอบ**

1. กด **Play (F5)**
2. เดินเข้าไปในบ้าน
3. ตรวจสอบว่าทุก Part ตรึงไว้แล้ว (ไม่หล่นหรือขยับ)
